### Project Title: ArchiLens - AR Measurement App Prototype

Description: A native iOS application project developed to explore the capabilities of ARKit and Swift. The app uses the device's camera to detect planes and measure distances between points in real-time.

Learning Goals:

Understanding the iOS App Lifecycle and UIKit components.

Implementing ARKit session management for spatial analysis.

Practicing Swift syntax and Object-Oriented Design patterns.

Tech Stack: Swift, ARKit, Xcode.

Current Status: Prototype phase (Active Learning).
